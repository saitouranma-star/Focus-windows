# Focus

Focusは、あなたの集中を助ける強力なデスクトップアプリです。特定のウェブサイト（SNSや動画サイトなど）へのアクセスを遮断し、作業効率を向上させます。


## 特徴

| 特徴 | 内容 |
| :--- | :--- |
| **強力なブロック** | hostsファイルを書き換え、アプリ停止中も制限を継続します。 |
| **一時解除タイマー** | 設定時間のみ制限を解除。終了後は自動で再ブロックします。 |
| **タスクトレイ常駐** | バックグラウンドで稼働し、作業を邪魔しません。 |


## インストール

お好みの方法でダウンロード・インストールできます。

### Releaseからダウンロード（推奨）

[こちらのページ](https://github.com/Nyanma11/Focus/releases)からダウンロード後、解凍してお使いください。

### Python環境をお持ちの方は、ソースコードから実行できます。

```cmd
# リポジトリをクローン
git clone https://github.com/あなたのユーザー名/Focus.git

# 必要なライブラリをインストール
pip install customtkinter pillow pystray plyer

# 実行
python focus.py
```

## 使い方

1. **アプリの起動**

   `Focus.exe` を起動します。hosts ファイルを書き換えるため、起動時に管理者権限の承認が必要です。起動した瞬間からブロックが有効になります。
   >※起動直後はタスクトレイ（画面右下の「^」の中）に隠れて常駐します。

2. **設定画面（GUI）の呼び出し方**

   設定を変更したり、一時解除タイマーを開始したいときは、タスクトレイにある「Focusのアイコン」をクリックしてください。ウィンドウが表示されます。

3. **ブラックリストの設定**

   「設定」タブから、制限したいサイトのURL（例：`youtube.com`）を入力して追加します。

4. **制限の一時解除**

   休憩したいときはタイマー時間を入力して**[一時解除スタート]**をクリックします。タイマー作動中のみ制限が解除されます。

5. **アプリの終了**

   タスクトレイ常駐アプリのためウィンドウを閉じてもアプリは終了しません。アプリを終了したいときはタスクトレイ内のアイコンを右クリックし、**[終了]**をクリックします。

   >※アプリを終了しても制限は解除されない仕様となっています。


## 注意事項

- 本アプリはシステムファイル（hosts）を編集します。
- 本アプリは、強力な集中環境を作るために **アプリを閉じてもブロックが継続する** 仕様になっています。
- 本アプリをアンインストールした際は、下記の **緊急時の手動解除方法** を参照してhostsファイルをクリーンアップしてください。


## 緊急時の手動解除方法

アプリが起動できない、または削除してしまった等の理由でアクセス制限が残ってしまった場合は、以下の手順で制限の解除が可能です。

1. 管理者権限でコマンドプロンプトを起動

2. 以下のコマンドを入力し、実行（※ hostsファイルが初期化されます）

```cmd
echo # Empty Hosts > C:\Windows\System32\drivers\etc\hosts
```
